# Flutter Auth UI

A simple authentication UI built using Flutter.  
This project contains Login and Signup screens with a modern gradient design and reusable widgets.

---

## Features

- Login screen UI
- Signup screen UI
- Gradient background design
- Reusable widgets (text fields, buttons, cards)
- Navigation between Login and Signup screens
- Clean and responsive layout

---

## Project Structure

- main.dart → App entry point  
- auth_ui.dart → Full UI implementation  

---
